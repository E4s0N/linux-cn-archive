陷入困境的 15 岁少年：Windows Vista

February 04, 2022

这并不是微软所承诺的……但也不全是坏事。

2007 年 1 月 30 日，也就是 15 年前，一再延期的 Windows Vista 发布了。

我们可以尽情地谈论 Vista 有多糟糕、人们有多讨厌它。它是有史以来最不受欢迎的版本之一，与 Windows ME 一样。但是，你今天运行的随便哪个现代 Windows，它都是 Vista 的后裔。

Vista 并没有微软像 2000 年承诺的那样，Windows “Whistler” 将合并 Windows 9x 和 Windows NT：这变成了 XP，每个人都记得 XP —— 基本上都是怀念。Windows Blackcomb 是 XP 的计划中的继任者。它从未面世。我们得到的是 Windows Longhorn。

Blackcomb 将有一个新的编程模型、一个新的用户界面，以及一个新的数据库驱动的存储引擎，“WinFS”……但它被延期了，然后被删除，最后被取消。

谈论 Vista 到底是什么更有意思。摆上货架的是技术上更保守的东西。

它有用户访问控制（UAC）：那些恼人的弹出式信息不断要求获得你的许可。（这实际上是微软意识到大部分的 Windows 机器都是独立的，没有连接到域。）

它有一个 3D 合成的用户界面，“Aero”，带有透明效果，尽管这确实使它变得有点慢。而当 Windows 7 推出时，人们很喜欢它 —— 所以微软将它从 Windows 8 中剥离了。（指的是外观，而不是合成过程；那仍然存在）。

它有一个包含“小部件”的侧边栏，这在 Windows 11 中又出现了。

尽管它在五年前就被干掉了，但 Vista 是 Windows 7 的基础。Vista 是 NT 的第 6 版；而 Windows 7 则是 Windows NT 6.1。

Windows 7 是经过调整的 Vista，响应速度更快。Windows 8 则是 Vista 2：是 Vista 的平板电脑版（它实际上相当不错）。

困难在于，人们已经习惯了台式机和鼠标，所以开始按钮又被迅速恢复了。Windows 8.1 则是 Vista 2.1。

Windows 10 是 Vista 3，Metro 现代应用程序被搁置，桌面应用程序再次成为中心。现在，Windows 11 删除了所有那些讨厌的、复杂的、混乱的东西，如垂直侧边栏。

Vista 是微软放弃其花哨的技术野心而开始简化事物的时候。现在有远见的人已经离开了。Surface 并没有彻底改变这个行业，它只是疏远了 OEM 厂商。

15 年过去了，微软仍在对 Vista 进行迭代。
